## Dataset
- **Source:** [NASA Exoplanet Archive](https://exoplanetarchive.ipac.caltech.edu/)  
- **Version:** 2025-09 release  
- **Description:** Contains confirmed exoplanets with parameters such as distance, radius, habitability score, and detection method.  
- **License:** Public (NASA Open Data)  
- **Note:** Dataset is not stored in this repository due to size. Download from the official link before running the project.  

---

##  Audio Generation
- **Tool:** [Google AI Studio](https://aistudio.google.com/generate-speech)  
- **Reason:** Audio recordings were generated using AI because we live in a crowded and noisy environment, making traditional recording unfeasible.
